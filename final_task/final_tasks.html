Final task. Perl + Javascript + AJAX

Create web-interface for your book storage. You should use AJAX to get a data from the server and make your own API. You should get a data without page reloading.

In Porta we use Apache as web-server, but for this task we write our own based on Mojolicious Lite web framework. You can install it from CPAN.

Server example and simple front-end can be found here. Download it and run on your PC.
